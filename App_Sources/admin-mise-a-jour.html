Paramètres système

Clé API Mistral (IA) et source des mises à jour (dépôt GitHub). La vérification et l'installation des mises à jour se font depuis la page Administration.

Clé API Mistral

La clé est optionnelle : vous pouvez générer les prompts et les copier dans un autre outil (ChatGPT, Claude, etc.) sans clé. Elle est requise uniquement pour les boutons « Générer le commentaire avec l'IA » (bilan et commentaire global d'évaluation). Créez une clé sur console.mistral.ai (plan gratuit disponible).

Les prompts ne contiennent aucune donnée confidentielle. Ce dispositif est aligné avec la charte d'utilisation des IA externes de l'Urssaf Caisse Nationale.
